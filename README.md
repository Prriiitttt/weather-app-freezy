# Freezy⚡️

A clean and simple weather app that shows you current weather conditions and hourly forecasts.

## What it does

Search for any city and get real-time weather info including temperature, wind speed, visibility, and humidity. Plus, see how the weather will change throughout the day with a 6-hour forecast.

The app also has a dark/light theme toggle.

## Built with

- HTML, CSS, JavaScript
- WeatherAPI for real-time data
- localStorage to save your theme preference

## Running it yourself

1. Clone or download this repo
2. Get a free API key from [WeatherAPI](https://www.weatherapi.com/)
3. Open `script.js` and replace `WEATHER_API` with your key
4. Open `index.html` in your browser

That's it! No build steps or installations needed.

## Features

- Search weather for any(mostly) city
- Live hourly forecast (updates every 4 hours)
- Dark/Light theme that persists
- Responsive design for mobile and desktop
